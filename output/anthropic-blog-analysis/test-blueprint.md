## Agent-First 战略

Anthropic 工程团队的**绝对主线**是 Agent 架构，贯穿 2024-2026 整个时间线。

从「简单可组合模式」的设计哲学出发，一路演进到 16 个并行 Claude 实例自主构建 C 编译器。

> 最成功的实现往往采用简单、可组合的模式，而非复杂框架。
